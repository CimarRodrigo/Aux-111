# Aux-111 — Introduction to Programming

Teaching material I wrote as a teaching assistant for **INF-111 (Introduction to Programming)** at Universidad Mayor de San Andrés, La Paz, Bolivia.

These are the exercises and examples I used in lab sessions with first-semester students — most of them writing their first lines of code. Everything is in Java, kept deliberately simple, and commented for people who are still building intuition rather than looking up syntax.

> The material is written in Spanish, since that's the language it was taught in.

---


## How to use it

Each folder holds standalone `.java` files. Nothing to install beyond a JDK:

```bash
javac Ejemplo.java
java Ejemplo
```

---

## Why it's public

Because good introductory material in Spanish is harder to find than it should be. If you're teaching a similar course, or learning on your own, take whatever's useful.

---

## A note on style

The code here optimizes for **readability over cleverness**. There are places where a one-liner would do the job and I wrote five lines instead, because a student reading it for the first time should be able to follow every step. That's intentional — please don't take it as idiomatic Java for production code.
